The agent loop

Five steps from idea to deployed app

Every prompt flows through the same orchestration cycle — no black boxes, just real code you own.

01

Spec

Parse intent, constraints, and scope from your natural-language prompt.

02

Context

Snapshot the file tree, block manifest, and recent conversation history.

03

Plan

Classify work: block activation, configuration, or feature generation. Review diffs in Plan mode.

04

Build

Apply patches, run TypeScript + Vite build gate, deploy backend via Supabase CLI.

05

Preview

Live iframe preview updates. Iterate with feedback until it ships.

Platform architecture

Frozen blocks, not regenerated boilerplate

Cross-cutting infrastructure is human-reviewed once. The agent activates blocks and generates domain code only.

🔐

Auth & RLS

Supabase Auth with profiles, JWT sessions, and Row Level Security on every user-owned table.

📦

Block registry

Deterministic activation recipes flip blocks from stub → enabled without LLM rewriting internals.

⚡

Edge functions

Secrets stay server-side. AI gateway, admin metrics, and domain logic run in Deno red zone.

📊

Admin dashboard

Block-aware metrics UI with dynamic tabs — Users, Storage, AI usage — gated by RBAC.

🌱

Plant Pal demo

Reference app: upload a plant photo, get AI health analysis, view history — full E2E pattern.

🚀

Free-tier deploy

Vercel Hobby + Supabase free. Export scaffold as standalone repo anytime.
